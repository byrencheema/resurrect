# Video Colorization Test Inputs Reference

A curated collection of freely downloadable black-and-white video clips and photographs
for testing and demonstrating video colorization. All sources are public domain or
freely usable, and require no login to download.

## 1. Charlie Chaplin Clips

Chaplin's pre-1927 work is in the public domain in the United States.

### 1.1 Charlie Chaplin Grand Archive (67 films, 1914-1967)

- **Page:** https://archive.org/details/sircharliechaplin
- **Description:** Comprehensive collection of 67 Chaplin films in H.264 MP4. Includes
  early Keystone, Essanay, Mutual, and First National shorts. Excellent variety of
  scenes: slapstick comedy, street scenes, interiors, crowd shots.
- **Format:** MP4 (H.264)
- **Size:** 22.9 GB total
- **Notable files for colorization testing:**
  - `Chaplin [1914-10] The Star Boarder.mp4` (57 MB) -- indoor/outdoor scenes
  - `Chaplin [1914-11] Mabel At The Wheel.mp4` (113 MB) -- outdoor racing scenes
- **Direct download pattern:**
  `https://archive.org/download/sircharliechaplin/{FILENAME}`

### 1.2 Charlie Chaplin Festival (4 shorts, 1917)

- **Page:** https://archive.org/details/charlie_chaplin_film_fest

  Easy Street, and The Immigrant. Good variety of indoor/outdoor scenes, people,
  and architecture.
- **Format:** MP4 (MPEG4 512Kb), OGG, DIVX
- **Size:** 3.7 GB total
- **License:** Public Domain

### 1.3 Making A Living (1914) -- Chaplin's First Film

- **Page:** https://archive.org/details/CC_1914_02_02_MakingALiving

## 2. Other Silent Era Films

### 2.1 Buster Keaton

#### The General (1926) -- RECOMMENDED

- **Page:** https://archive.org/details/The_General_Buster_Keaton
- **Alt page:** https://archive.org/details/BusterKeatonTheGeneral1926
- **Description:** One of the greatest silent films ever made. Civil War-era train
  chase film. Incredible variety of scenes: locomotives, bridges, forests, armies,
  buildings, uniforms. Outstanding for colorization demos.
- **Format:** MP4 (H.264, 512Kb MPEG4), OGG
- **Runtime:** ~78 minutes
- **License:** Public Domain (copyright not renewed in 1954)

#### Cops (1922)

#### Go West (1925)

## 5. How to Download from Archive.org

### Web Browser Method

1. Go to the item page (e.g., https://archive.org/details/The_General_Buster_Keaton)
2. Look for "DOWNLOAD OPTIONS" on the right sidebar
3. Click the format you want (e.g., "MPEG4", "H.264", "OGG VIDEO")
4. The file will download directly

### Direct URL Construction

Archive.org direct download URLs follow this pattern:

```
https://archive.org/download/{ITEM_IDENTIFIER}/{FILENAME}
```

The ITEM_IDENTIFIER is the last part of the `/details/` URL. For example:
- Page: `https://archive.org/details/The_General_Buster_Keaton`
- Identifier: `The_General_Buster_Keaton`
- Download: `https://archive.org/download/The_General_Buster_Keaton/{filename}.mp4`

To find exact filenames, visit:
```
https://archive.org/download/{ITEM_IDENTIFIER}/
```
This shows all available files with sizes and formats.

### Recommended Formats for Colorization Work

| Format | Extension | Quality | Best For |
|--------|-----------|---------|----------|
| H.264 | .mp4 or .ia.mp4 | High | Primary working format |
| MPEG4 512Kb | _512kb.mp4 | Low | Quick previews/testing |
| Original | varies | Highest | Best quality source |

## 6. How to Download from Library of Congress

### Individual Images

1. Go to the item page (e.g., https://www.loc.gov/pictures/item/2017762891/)
2. Click on the image to see available sizes
3. Look for download links: "JPEG (XXkb)", "TIFF (XX MB)"
4. Click the desired format to download

### TIFF Downloads

For the highest resolution, always choose the TIFF option. These are typically
scanned from original negatives at archival quality (often 4000+ DPI).

- Small JPEG: ~50-150 KB (good for previews)
- Large JPEG: ~150-500 KB (good for web use)
- TIFF: ~10-500 MB (full archival resolution, best for colorization)

### Bulk Downloads

The Library of Congress provides bulk data access via:
- **loc.gov JSON API:** https://www.loc.gov/apis/
- Individual collection pages often have "Download" options for batches

## 7. Bulk Download with Command Line

### Download a Single File from Archive.org

```bash
# Using wget
wget "https://archive.org/download/The_General_Buster_Keaton/The_General_Buster_Keaton.mp4"

# Using curl
curl -L -O "https://archive.org/download/The_General_Buster_Keaton/The_General_Buster_Keaton.mp4"
```

### Download All Files from an Archive.org Item

```bash
# Using wget to mirror an entire item
wget -r -np -nH --cut-dirs=2 -R "index.html*" \
  "https://archive.org/download/The_General_Buster_Keaton/"

# Download only MP4 files
wget -r -np -nH --cut-dirs=2 -A "*.mp4" \
  "https://archive.org/download/The_General_Buster_Keaton/"
```

### Download from Library of Congress

```bash
# Download a specific TIFF (Migrant Mother)
wget "https://tile.loc.gov/storage-services/service/pnp/ppmsca/12800/12883v.tif" \
  -O migrant_mother.tif

# Download a JPEG version
wget "https://tile.loc.gov/storage-services/service/pnp/fsa/8b29000/8b29500/8b29516v.jpg" \
  -O migrant_mother.jpg
```

### Using ia (Internet Archive command-line tool)

```bash
# Install the Internet Archive CLI tool
pip install internetarchive

# Download a specific item
ia download The_General_Buster_Keaton --format "MPEG4"

# Download only the h264 version
ia download The_General_Buster_Keaton --format "h.264"

# List available files for an item
ia list The_General_Buster_Keaton
```

## Notes

- **Copyright:** All items listed here are believed to be in the public domain in the
  United States. Copyright status may vary in other jurisdictions. Always verify
  before commercial use.
- **Attribution:** While not legally required for public domain works, crediting the
  source (e.g., "Source: Internet Archive" or "Source: Library of Congress") is
  good practice and helps others find these resources.
- **Quality:** For the best colorization results, always download the highest
  resolution version available. On Archive.org, prefer H.264 or original format
  over the 512Kb MPEG4 versions.
- **File sizes:** Full-length silent films in good quality are typically 500 MB to
  2 GB. Plan storage accordingly.
